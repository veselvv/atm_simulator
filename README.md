# 🏦 ATM Simulator

**C • GTK4 • cJSON • Linux**

---

ATM Simulator — это десктопное приложение для Linux, имитирующее работу банкомата.  
Проект написан на **C** с графическим интерфейсом на **GTK4** для пользователей и консольной программой для администратора.  
Все данные хранятся локально в **JSON-файлах** с использованием библиотеки **cJSON**.

---

## 🎯 Функционал

### 👤 Для пользователя (GTK4)
- Авторизация по номеру карты / телефону / логину
- Блокировка карты после 3 неудачных попыток ввода PIN
- Проверка баланса
- Снятие наличных
- Пополнение счёта
- Перевод средств между картами
- Просмотр истории операций
- Интуитивно понятный графический интерфейс

### 🛠 Для администратора (консоль)
- Создание новых карт
- Поиск карт по номеру, телефону, логину, имени
- Редактирование баланса и PIN-кода
- Блокировка / разблокировка карт
- Просмотр всех карт
- Сохранение данных в JSON

## 🚀 Установка и запуск

### 1. Установите зависимости

```bash
chmod +x install-deps.sh
./install-deps.sh
```
Или вручную (Ubuntu/Debian):

```bash
sudo apt update
sudo apt install build-essential libgtk-4-dev pkg-config valgrind make
```
### 2. Соберите проект

```bash
make
```
Makefile автоматически создаст папки data/ и logs/.

### 3. Запустите приложение

```bash
# Графический интерфейс (из корня проекта)
make run
# или
cd build && ./myapp

# Административная консоль
make run-admin
# или
cd build && ./admin
```

---

## 🔧 Команды Makefile

| Команда | Описание |
|-----------|------------|
| **make** | Собрать всё (myapp + admin) |
| **make run** | Запустить GUI |
| **make run-admin** | 	Запустить админку |
| **make clean** | Очистить объектные файлы и бинарники |
| **make distclean** | Удалить папку build/ |
|**make clean-data**| Удалить data/ и logs/ |
|**make rebuild**| Полная пересборка |

## 📂 Хранение данных

**data/test_database.json** — основная база данных (карты, балансы, PIN-коды)

**logs/operations_XXXXX.json** — история операций для каждой карты

---

## 🔐 Безопасность

- PIN-код хранится в открытом виде (учебный проект, для демонстрации)
- Карта блокируется после 3 неудачных попыток ввода PIN
- Все операции логируются с указанием времени и суммы
